# Project Summary
The SA'DA HOLDINGS project develops a corporate website for a Saudi-based holding group headquartered in Al-Khobar. It showcases a diversified portfolio of high-value companies across sectors including desalination, water management, laboratory solutions, IT infrastructure, AI-driven technologies, digital transformation, sports event mobilization, and selective passion projects. This platform enhances stakeholder engagement and reflects the company's commitment to quality and trust while emphasizing its active presence throughout the Kingdom, particularly in Al-Khobar, Riyadh, and Jeddah. A PDF company profile has been created to match the website's design, providing a comprehensive overview of the company and its offerings.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for modern web applications.
- **Lucide Icons**: Icon set for user interface elements.
- **WeasyPrint**: Python library for generating PDFs from HTML/CSS.

# Usage
To install dependencies, build, and run the project:
1. Navigate to the project directory: `cd /workspace/app/frontend`
2. Install dependencies: `pnpm install`
3. Run the development server: `pnpm run dev`
4. To generate the PDF company profile, ensure Python and WeasyPrint are installed, then run `python3 generate_profile_pdf.py` from the `public` directory.
