# Project Summary
The SA'DA HOLDINGS project is focused on creating a corporate website for a diversified Saudi holding group. This platform highlights the group's extensive portfolio across various sectors, such as desalination, water management, laboratory solutions, IT infrastructure, and AI technologies. The website aims to enhance stakeholder engagement by showcasing the company's commitment to quality, trust, and purpose.

# Project Module Description
- **Navbar**: Navigation links to various sections with the updated SA'DA logo.
- **Hero**: A cinematic introduction featuring the tagline "From Science to Transformations."
- **About**: Clarifies the meaning of SA'DA and the company's refined philosophy.
- **Vision & Mission**: A unified statement emphasizing the creation and growth of purposeful businesses that address real market needs through quality and innovation.
- **Companies**: Highlights the different companies under SA'DA HOLDINGS and their unique offerings, including updated focus for Artland Infotech.
- **Ecosystem**: Displays the interconnection of businesses within the SA'DA group.
- **Impact**: Showcases key metrics and achievements.
- **Why SA'DA**: Details the guiding principles for the company's operations.
- **Leadership**: Presents the leadership team and their vision.
- **Contact**: A form for potential partners to reach out.
- **Footer**: Contains links to company information and updated navigation.

# File Description Inventory
- **README.md**: Documentation for project setup and usage.
- **components/**: Contains all reusable React components, including updates to About and Vision & Mission, and modifications in Companies.
- **pages/**: Contains the main pages of the application.
- **public/**: Holds static files like images and icons, including the updated SA'DA logo and the new Brown Truck image.
- **src/**: Source files for application logic and styles.
- **index.css**: Main stylesheet for global styles.
- **package.json**: Lists project dependencies and scripts.
- **tailwind.config.ts**: Configuration file for Tailwind CSS.
- **vite.config.ts**: Configuration file for Vite.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for modern web applications.
- **Lucide Icons**: Icon set for user interface elements.

# Usage
To install dependencies, build, and run the project:
1. Navigate to the project directory: `cd /workspace/app/frontend`
2. Install dependencies: `pnpm install`
3. Run the development server: `pnpm run dev`
