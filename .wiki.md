# Project Summary
The SA'DA HOLDINGS project is focused on creating a corporate website for a diversified Saudi-based holding group headquartered in Al Khobar. The website showcases the group's extensive portfolio, which includes sectors such as desalination, water management, laboratory solutions, IT infrastructure, AI-driven technologies, digital transformation, sports event mobilization, and selected passion projects. The website aims to embody the company's values of quality, trust, and purpose.

# Project Module Description
- **Navbar**: Provides navigation links to different sections of the website, featuring the updated SA'DA logo.
- **Hero**: Displays a cinematic introduction with a prominent message about SA'DA's mission and values, now reflecting specific sectors of focus.
- **About**: Introduces the meaning of SA'DA and the company's refined philosophy.
- **Vision & Mission**: Outlines the core vision and mission statements of the company, aligned with the updated description.
- **Companies**: Showcases the various companies under SA'DA HOLDINGS, highlighting their services and unique selling points.
- **Ecosystem**: Illustrates the interconnectedness of the businesses within the SA'DA group, reflecting the updated sectors.
- **Impact**: Displays key metrics and achievements of the company.
- **Why SA'DA**: Explains the principles guiding the company's operations and strategy.
- **Leadership**: Introduces the leadership team and their vision for the company.
- **Contact**: Provides a form for potential partners to reach out and collaborate.
- **Footer**: Contains links to company information, navigation, and contact details, now featuring the updated logo.

# File Description Inventory

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for modern web applications.
- **Lucide Icons**: Icon set for user interface elements.

# Usage
To install dependencies, build, and run the project:
1. Navigate to the project directory: `cd /workspace/app/frontend`
2. Install dependencies: `pnpm install`
3. Run the development server: `pnpm run dev`
