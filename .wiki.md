# Project Summary
The SA'DA HOLDINGS project is focused on developing a corporate website that showcases a Saudi-based business group. The site emphasizes the group's diverse portfolio, which includes companies in water treatment, technology, scientific solutions, branding, event mobilization, and lifestyle food concepts. The website aims to reflect the company's values and mission: to build businesses that matter, driven by quality, trust, and purpose.

# Project Module Description
- **Navbar**: Provides navigation links to different sections of the website.
- **Hero**: Displays a cinematic introduction with a prominent message about SA'DA's mission and values.
- **About**: Introduces the meaning of SA'DA and the company's philosophy.
- **Vision & Mission**: Outlines the core vision and mission statements of the company.
- **Companies**: Showcases the various companies under SA'DA HOLDINGS, highlighting their services and unique selling points.
- **Ecosystem**: Illustrates the interconnectedness of the businesses within the SA'DA group.
- **Impact**: Displays key metrics and achievements of the company.
- **Why SA'DA**: Explains the principles guiding the company's operations and strategy.
- **Leadership**: Introduces the leadership team and their vision for the company.
- **Contact**: Provides a form for potential partners to reach out and collaborate.
- **Footer**: Contains links to company information, navigation, and contact details.

# File Description Inventory

- **index.css**: Main stylesheet for global styles.

- **vite.config.ts**: Configuration file for Vite.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for modern web applications.
- **Lucide Icons**: Icon set for user interface elements.

# Usage
To install dependencies, build, and run the project:
1. Navigate to the project directory: `cd /workspace/app/frontend`
2. Install dependencies: `pnpm install`
3. Run the development server: `pnpm run dev`
