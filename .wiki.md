# Project Summary
The SA'DA HOLDINGS project develops a corporate website for a diversified Saudi holding group, showcasing its extensive portfolio in sectors such as desalination, water management, laboratory solutions, IT infrastructure, and AI technologies. The platform enhances stakeholder engagement and reflects the company's commitment to quality, trust, and purpose, while also highlighting its active presence across all provinces in the Kingdom, particularly in Al-Khobar, Riyadh, and Jeddah.

# Project Module Description
- **Navbar**: Navigation links to various sections with the updated SA'DA logo, excluding the Ecosystem tab.
- **Hero**: A cinematic introduction featuring the tagline "From Science to Transformations."
- **About**: Clarifies the meaning of SA'DA, stating, "The name SA'DA, سعادة, means happiness, sits at the heart of our business philosophy."
- **Vision & Mission**: A unified statement emphasizing purposeful business creation and growth through quality and innovation.
- **Companies**: Highlights the different companies under SA'DA HOLDINGS and their unique offerings, including "Classic Bufiya sandwiches" under Brown Truck.
- **Impact**: Showcases key metrics and achievements.
- **Why SA'DA**: Details the guiding principles for the company's operations with text adjustments for clarity.
- **Leadership**: Presents the leadership team and their vision.
- **Contact**: A form for potential partners to reach out, updated to include the phone number 920022569 and the email contact@sadaholdings.com, along with information about SA'DA HOLDINGS offices across the Kingdom.
- **Footer**: Contains links to company information, updated navigation reflecting the new email address and phone number, and mentions the company's active presence in all provinces.

# File Description Inventory
- **README.md**: Documentation for project setup and usage.
- **components/**: Contains all reusable React components, including updates to About, Vision & Mission, Companies, Why SA'DA, Contact, and Footer for office locations and phone number.
- **pages/**: Contains the main pages of the application, excluding the Ecosystem section.
- **public/**: Holds static files like images and icons, including the updated SA'DA logo and the new Brown Truck image.
- **src/**: Source files for application logic and styles.
- **index.css**: Main stylesheet for global styles.
- **package.json**: Lists project dependencies and scripts.
- **tailwind.config.ts**: Configuration file for Tailwind CSS.
- **vite.config.ts**: Configuration file for Vite.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for modern web applications.
- **Lucide Icons**: Icon set for user interface elements.

# Usage
To install dependencies, build, and run the project:
1. Navigate to the project directory: `cd /workspace/app/frontend`
2. Install dependencies: `pnpm install`
3. Run the development server: `pnpm run dev`
